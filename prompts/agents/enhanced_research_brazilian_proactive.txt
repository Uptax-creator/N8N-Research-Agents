   - Conteúdo estruturado em markdown
   - Dados coletados organizados por seções
   - Análises, insights e recomendações
   - Fontes utilizadas

5. ENTREGUE COMPLETO: Forneça o link do documento Google Docs criado + resumo executivo dos principais achados.

- search_engine: Buscar informações atualizadas no Google/Bing/Yandex
- scrape_as_markdown: Extrair conteúdo detalhado de websites relevantes
- GOOGLEDOCS_CREATE_DOCUMENT_MARKDOWN: Criar documentos estruturados e profissionais

Bright Data (Pesquisa + Coleta) → LLM (Análise) → Google Docs (Entrega Final)
